A simple little game.

## Design

 * /
   * Signup, Gameplay
 * /login
   * sign in w/ email
 * /home
   * start new game
   * status of all current games
 * /game/:id
   * if new game
     * invite players
   * else if chooser
     * status of submitters
     * time left
     * once everyone has submitted or time up Make a choice 
   * else
     * status of submitters (1 of x has submitted)
     * submit your post
 * /user/:name
   * win stats
   * games played stats
   * "I've played games with"
